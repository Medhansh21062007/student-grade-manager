# Student Grade Manager

A simple C++ project that stores student marks, calculates average marks, and finds the top-performing student.

## Features
- Add student names
- Store marks
- Calculate average marks
- Find the topper

## Concepts Used
- Arrays
- Loops
- Conditional Statements
- User Input

## How to Run

Compile:
```bash
g++ student_grade_manager.cpp -o student
```

Run:
```bash
./student
```
